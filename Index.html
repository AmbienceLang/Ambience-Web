Ambience

Simple Syntax

Print

Print normal on the current line: 

out("your text here")

Print normal on the next following line: 

out!("your text here")

If

Simple if clause with Ambience: 


init cond1 = 2
init cond2 = 4

if (cond1 < cond2) {
    out("{$} is more then {$}", cond2, cond1)
}

While

Simple While Loop with Ambience: 


init cond1 = true;
init loopcount = 0;

while (cond1 == true) {
    loopcount += 1;
}



//Progressed While Loop with Ambience


init cond1 = true;
init loopcount = 0;

while (cond1 == true) {
    loopcount += 1;
    while (loopcount > 5){
        out!("Already looped more then {$} times", loopcount); // newline
        out("Already looped more then {$} times", loopcount); // not newline
    }
    if (loopcount !< 20) { // if loopcount is not smaller then 20
        cond1 = false;
    }
}
